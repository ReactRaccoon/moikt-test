# Тестовое задание для МО ИКТ

____

## Порядок запуска:

#### Установка зависимостей

 В дириектории проекта:

```
  yarn install
```

#### Запуск приложения

```
 yarn start 
```

#### Использование

Доступ к приложению находится по адресу:

```
localhost:3000
```
